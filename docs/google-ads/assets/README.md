# Google Ads marketing assets

Image extensions for the Always-On Local Search 2026 campaign. Generated 2026-05-12 with
Higgsfield `marketing_studio_image`. Briefs and assignment to ad groups live in
`../always-on-search-2026-setup.md` and `../always-on-search-2026-README.md`.

These files are NOT served by the website — they are stored here so the team can reference
exactly what was uploaded to Google Ads, and so future A/B test variants can iterate from a
known baseline.

## Refreshing for A/B tests

Re-run the briefs in `../always-on-search-2026-README.md` §3 with Higgsfield. Iterate on prompts
based on what the existing extensions show as winners (CTR data in Google Ads after 30 days).
Save new variants here with a date suffix, e.g. `01-hero-bouquet-square-2026-08.png`.

## Important constraints

- Never use AI-generated images for Google Business Profile photos. GBP detects synthetic
  imagery and penalizes the listing. Use real iPhone photos for GBP.
- Never claim "free delivery" in any image overlay or accompanying text. Diva Flowers charges
  per-zone fees ($10–$30); use "Same-Day Delivery from $10" or omit delivery messaging.
